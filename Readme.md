# EUResto

Android приложение (Jetpack Compose) за бързо смятане на ресто между евро и лева по курс 1 EUR = 1.95583 лв.

## Основни функции
- Поле за въвеждане на сметката в евро.
- Автоматично показване на стойността в лева.
- Полета „Платено“ за лева и евро с цифрова клавиатура.
- Автоматично изчисляване на рестото едновременно в лева и евро.

## Изисквания за компилиране
- Android Studio Giraffe/по-нова.
- Android Gradle Plugin 8.4+ и JDK 17.
